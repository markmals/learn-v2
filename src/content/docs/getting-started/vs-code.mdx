Visual Studio Code is the standard editor/IDE for modern web development. It’s built on [Electron](https://www.electronjs.org), so it runs the same on macOS, Windows, and Linux.

VS Code is [open source](https://github.com/microsoft/vscode) and highly extensible, with a [marketplace of thousands of extensions](https://marketplace.visualstudio.com).

## Download & Install

To download VS Code, open Terminal and run this command to install using Homebrew:

```sh
brew install --cask visual-studio-code
```

VS Code comes with a command line program (`code`), which allows you to script actions and open VS Code from the command line. To enable the `code` command:

1. Open VS Code
2. Open the Command Palette using `⌘ + shift + P`
3. Select `Shell Command: Install 'code' command in PATH`
4. Quit and restart VS Code and Terminal

## Familiarizing Yourself with the Editor

Here’s a quick tour of the core UI.

### Explorer

- Shows files/folders. Right‑click to create files. Drag to move.
- Use `⌘ + P` to "Go to File" quickly.
- Built‑in file nesting groups related files (configured in this repo).

### Search

- Global text search across the workspace.
- Use regex and case/word match toggles.

### Source Control

- Integrated Git view. Stage, commit, and push without leaving the editor.
- Sign in to GitHub to use "Pull Requests" extension for reviews.

### Testing

- The Vitest extension shows tests, watch mode, and inline results for framework code.

### Extensions

- Manage installed extensions and settings.

## Integrated Terminal

VS Code comes with a built-in terminal so you can run commands in the command line without leaving the context of your current project. 

You can open the integrated terminal by clicking on the "Toggle Panel" button in the upper right-hand corner of the VS Code editor (or by pressing `⌘ + J`) and selecting the "Terminal" tab.
